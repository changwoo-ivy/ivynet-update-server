1.1.0-alpha.5
=============
* github 연동 기능 강화.
  이번 버전부터 각 사용자마다 github user / private access token 필드에 값을 채울 수 있다.
  그리고 각 프로젝트마다 각 사용자를 지정할 수 있으며, 해당 사용자는 Github 리포지터리에 접근할 수 있다.
  이렇게 하여 리포지터리가 private 상태라도 문제없이 접근하여 태그되었을 때 파일을 다운로드 받을 수 있다.

* github webhook 정책
  * 중복된 태그 이름이 있는 경우, 현재의 웹훅이 주는 내용으로 변경된다.
  * 릴리즈 포스트 삭제시, 첨부 파일도 지워진다.
  * 태그를 지우면 릴리즈도 지워진다.
  * 프로젝트의 최신 버전은 수동으로 지정한 것과는 관계없이 항상 현재 릴리즈에서 인식할 수 있는 가장 높은 버전으로 고정된다.
